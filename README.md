# 📝 Simple Text Editor

A simple desktop-based **Text Editor** built using **Python** and **Tkinter**. It provides an easy-to-use graphical interface for creating, editing, opening, and saving text files.

---

## 📖 Project Overview

The **Simple Text Editor** is a desktop application developed using Python's Tkinter library. It is designed to perform basic text editing operations through a clean and intuitive interface. The application allows users to create new documents, open existing text files, edit their content, and save changes. This project demonstrates the fundamentals of GUI development, file handling, and event-driven programming in Python.

---

## ✨ Features

- 📄 Create a new text file
- 📂 Open existing text files
- 💾 Save text files
- ✍️ Edit text easily
- 🖥️ Simple and user-friendly interface
- ⚡ Lightweight and fast

---

## 🚀 How to Run

1. Clone the repository

```bash

```

2. Navigate to the project directory

```bash
cd Simple-Text-Editor
```

3. Run the application

```bash
python main.py
```
